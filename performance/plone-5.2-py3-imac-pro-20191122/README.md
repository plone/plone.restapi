# REST API PERFORMANCE TESTING

## Setup

Install jMeter:

````
brew install jmeter
````

Setup Buildout:

````
make build-plone-5.2-performance
````

Start Plone instance:

````
bin/instance fg
````

Run performance test:

````
make test-performance
````
